# Data-repository-scraping-and-dynamical-downloading-locally-No-hard-coded-path-needed
Function to seamlessly scrape the web for your dataset and efficiently download it to dynamically created directory locally-No hard coded path needed. The function seamlessly handles any 500-level temporary errors (500, 502, 503 &amp; 504)

Jupyter notebook: Function seamlessly scrape the web for your dataset and efficiently downloads it to a dynamically created directory locally. It handles 500-level temporary errors (500, 502, 503 & 504). 

Developed to seamlessly scrape the datasets from websites (e.g., NCEP) and efficiently download them to a dynamically created local directory. This not only automates the data acquisition process but also streamlines the organisation of your files and directory, avoiding overwriting. This will significantly enhance your productivity so you can focus on analysis rather than cumbersome manual data handling.
